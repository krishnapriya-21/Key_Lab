--------------------------------------------------------TYPING-GAME------------------------------------------------------

Here is a Simple Typing Test, which generates a random sentence upon start.

The user can type the displayed sentence in the Text Box.

1. Upon Input of a Wrong Character, the  User will be thrown an error warning.

2. Upon Successful Completion of the Sentence, the User Will be notified with
   Total Time Taken to complete the sentence in Seconds .

Additional Enhanced Interactivity Feature :

Audio Effect on Error and Success.
